# 🚀 Quick Start — Protegrity Data Protection for Cursor

Welcome. This guide helps you get started with Protegrity AI Developer Edition in Cursor in a simple, customer-driven way.

---

## Step 1: Pick one task
Choose the task you want Cursor to help with:

- Find sensitive data
- Redact or mask text
- Protect or unprotect text
- Scan conversation risk
- Generate synthetic data
- Anonymize structured data

---

## Step 2: Install only what you need

- For discovery, redaction, guardrails, synthetic data, or anonymization: install and start the required Docker service
- For protect/unprotect: create a Developer Edition account, set the environment variables, and install the SDK

If you only need one feature, stop after that feature is ready.

---

## Step 3: Use the commands in Cursor

Open the Command Palette (`Cmd+Shift+P` on Mac, `Ctrl+Shift+P` on Windows/Linux) and type **Protegrity**.

Recommended commands:

- `Protegrity: Setup Wizard`
- `Protegrity: Status Check`
- `Protegrity: Analyze Data Sensitivity`
- `Protegrity: Protect Text`
- `Protegrity: Redact Sensitive Data`
- `Protegrity: Scan Conversation Risk`
- `Protegrity: Generate Synthetic Data`
- `Protegrity: Anonymize Data`

---

## Step 4: Verify it works

Use **Protegrity: Status Check** and confirm:

- The plugin is loaded
- Your required service is reachable
- Your credentials are set if you need tokenization
- You are ready to run the task you selected

---

## Need help?

If you are new to Developer Edition, go back to the setup wizard and choose one feature at a time.
