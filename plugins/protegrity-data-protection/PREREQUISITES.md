# Prerequisites

> NOTE: MCP references removed from this repository. The plugin now calls Protegrity APIs directly. To use the plugin, either run the Protegrity Developer Edition services locally (Docker, exposing ports 8580 and 8581) or provide cloud credentials via the DEV_EDITION_EMAIL, DEV_EDITION_PASSWORD and DEV_EDITION_API_KEY environment variables.

This document describes the feature-specific prerequisites and setup steps required to make the Protegrity Data Protection plugin work with the Protegrity AI Developer Edition APIs.
