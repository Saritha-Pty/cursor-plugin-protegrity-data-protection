# Synthetic Data Generation

Use this skill to create realistic, anonymized datasets for safe development and testing.

## When to use

- During model training and development
- For secure testing workflows
- When sharing datasets across teams
- For AI model evaluation without privacy risk
- When data volume is insufficient for model development

## Instructions

1. Provide a sample of real data or data schema.
2. Specify the number of synthetic records to generate.
3. Configure statistical properties to preserve (distributions, relationships).
4. The API generates synthetic data maintaining data patterns without exposing originals.
5. Use the synthetic dataset for development and testing.

## Features

- Preserves statistical properties and relationships
- Maintains data type and format consistency
- Supports categorical and numerical features
- Customizable preservation of key characteristics
- Full audit trail of generation parameters

For detailed usage, refer to the [Synthetic Data documentation](https://docs.protegrity.com/synthetic-data/1.0.0/docs/).
