# Synthetic Data Generation

Use this skill when the customer wants safe test data that resembles production data without exposing real values.

## Customer goal

Help the customer create realistic sample data for development, testing, or demos.

## When to use

- For test environments
- For demos and prototypes
- When real data should not be used

## What the customer needs

- Docker Desktop
- Protegrity AI Developer Edition synthetic data service

## What this skill should do

1. Ask the customer for a schema or example data.
2. Ask how much synthetic data is needed.
3. Generate the synthetic output.
4. Explain what was created in simple terms.

## Guidance

- Keep the dataset useful for testing.
- Do not expose real sensitive values.
